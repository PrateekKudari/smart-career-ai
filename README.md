# Smart Career AI 🚀

An AI-powered Career Recommendation and Resume Analysis System built using Machine Learning, NLP, Flask, and Data Analytics.

---

# Features

✅ Resume Classification  
✅ Resume Score Prediction  
✅ Skill Extraction using NLP  
✅ AI Job Recommendation System  
✅ Job Match Percentage  
✅ Analytics Dashboard  
✅ Charts & Visualizations  
✅ Resume Upload (PDF)  
✅ Machine Learning Models  
✅ Modern Bootstrap UI  
✅ Flask Web Application  
✅ Deployable on Render

# Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/smart-career-ai.git
```

## Move Into Folder

```bash
cd smart-career-ai
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

# Run Project

```bash
python app.py
```

Open Browser:

```bash
http://127.0.0.1:5000
```

---

# ML Features

## Resume Classification
Predicts resume category using NLP and Machine Learning.

## Resume Score
Calculates resume quality score.

## Skill Extraction
Extracts technical skills from resume text.

## AI Job Recommendation
Recommends jobs using cosine similarity.

## Match Percentage
Shows how strongly resume matches jobs.
